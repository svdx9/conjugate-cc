# Conjugate.cc Frontend

The frontend is a SolidJS application for practicing verb conjugations.

## Repository Layout (relative to /frontend)

- `src/`: Source code.
- `public/`: Static assets.

## Prerequisites

- Node.js v20+
- npm (or pnpm/yarn)

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application uses Vite for building and development.

### Linting and Formatting

```bash
npm run lint
npm run format
```
